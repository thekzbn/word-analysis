# Top 5000 Words: Documentation
**Author**: Ayomide Deji-Adeyale (@thekzbn)

This documentation suite provides a technical and structural overview of the "Top 5000 Words" linguistic analysis application. It is designed for developers who wish to understand the underlying logic, component architecture, and design implementation.

---

## 1. Documentation Structure

-   [**Algorithm.md**](./Algorithm.md)  
    A line-by-line breakdown of the character frequency and pattern extraction logic. Start here if you want to know how the raw word list is transformed into data.
-   [**Home.md**](./Home.md)  
    Details the dashboard architecture, the Bento grid layout, and the high-level data aggregation.
-   [**Detail.md**](./Detail.md)  
    Explains the dynamic routing and the granular analysis modules, including the structural modal and comparative charts.

---

## 2. Technical Stack Overview
-   **Frontend**: React (v19) with TypeScript.
-   **Styling**: Tailwind CSS (Utility-first) following the "bored" design language.
-   **Data Processing**: Client-side JavaScript/TypeScript.
-   **Visualization**: Recharts (for bar graphs).
-   **Motion**: Framer Motion (for transitions and modals).

---

## 3. How to Use
These documents are written in standard Markdown. You can read them directly in any Git repository browser or a local Markdown viewer. Each file contains code snippets and logic justifications to assist in maintenance or expansion of the tool.
